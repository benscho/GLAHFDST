Welcome to the GLAHF Explorer! This interactive web mapping application is designed to present the data collected as part of the GLAHF project in an accessible and highly usable way.
Map navigation using mouse and keyboard:
  • Drag to pan
  • SHIFT + Click to recenter
  • SHIFT + Drag to zoom in
  • SHIFT + CTRL + Drag to zoom out
  • Mouse Scroll Forward to zoom in
  • Mouse Scroll Backward to zoom out
  • Use Arrow keys to pan
  • + key to zoom in a level
  • - key to zoom out a level
  • Double Click to Center and Zoom in
A variety of data layers can be enabled or disabled:
  • Clicking on a group causes it to expand
  • Individual layers can be enabled or disabled
  • Layers with a slider view are marked
The Criteria tool is designed to allow you to explore the GLAHF datasets in a more open-ended fashion. Define your criteria, and the tool will compute the intersection between them (if any exists).
The draw tool allows you to add your own markup to the map in the form of text or geometry.
The query tool allows you interact directly with the GLAHF data. Please let us know what types of data you would like to see made available!
The identify tool is designed to provide you with information about an active map layer. You can choose which layers you would like to select from.
The save tool allows you to save your currently active layers, drawing, map extent, and criteria to your browser's local storage so you can resume at a later time.
The print tool prepares the current map for printing, and provides you with a .PDF file.
The measure tool allows you to measure spaces on the map. Supported options include area, distance, and latitude/longitude coordinates.
The locate tool allows you to translate lat/long coordinates or an address to an actual point on the map.
